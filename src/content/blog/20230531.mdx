## 概要

この前ネットを見ていたら、偶然Astro v2.5の<a href="https://astro.build/blog/astro-250/" target="_blank" rel="noreferrer noopener">リリース記事</a>を発見。
前回のv2.2に関してはあまり旨味がないなということで見送ったのですが、今回の「HTML minification」と「Parallelized rendering」、これらは即効性がありそうだと思ったので上げてみることにしました。

「Data collections and references」もまだ使っていませんが、ますますCMSの代わりになりそうな気がします。
データの自動生成とかしても面白そうです。

## 手順

手順というほど書くことはないです。

```bash
npm i astro@latest
```

でバージョンを上げて、

```js:astro.config.mjs
export default defineConfig({
  compressHTML: true,
})
```

を追加するだけです。
